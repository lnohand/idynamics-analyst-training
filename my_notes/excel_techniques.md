Dynamic month labels with TEXT(DATE()) Instead of typing "Feb 2026" as static text, use =TEXT(DATE(2026,2,1),"mmm yyyy") to generate it as a formula. DATE() builds a real date serial from year/month/day inputs, and TEXT() formats it into a readable string using a format code — "mmm yyyy" gives "Feb 2026", "mmm-yy" gives "Feb-26", "mmmm yyyy" gives "February 2026". The advantage over hardcoding is that formula-driven labels stay in sync if the underlying data shifts, and you can make them fully dynamic by replacing the month number with a cell reference or wrapping in EDATE() to step forward one month at a time.

Cross-tab references To pull a value from another sheet, use the syntax ='Sheet Name'!B9 — single quotes are required whenever the sheet name contains spaces or special characters, and the exclamation mark always separates the sheet name from the cell address. The golden rule is: if a value already exists somewhere in the workbook, reference it rather than retyping it. In practice this means chaining months together (Feb's opening MRR references Jan's closing MRR cell, Mar references Feb, and so on) and pointing summary sheets like the KPI Tracker directly at the source A vs F sheets. References are relative by default, so add $ anchors (=$B$9) if you're copying the formula across rows or columns and need it to stay locked.
